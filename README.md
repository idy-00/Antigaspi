# Antigaspi - Revalorisation des Invendus Alimentaires

**Antigaspi** est une plateforme SaaS permettant aux commercants de vendre leurs invendus alimentaires a prix reduit.

## Fonctionnalites
- Geolocalisation des boutiques partenaires
- - Reservation de paniers en ligne
  - - Interface de gestion pour les commercants
    - - Notifications en temps reel
     
      - ## Technologies
      - - Laravel (PHP)
        - - MySQL
          - - Blade & Tailwind CSS
           
            - ## Installation
            - 1. Clonez le projet
              2. 2. composer install
                 3. 3. php artisan migrate
                    4. 4. php artisan serve
                       5.
